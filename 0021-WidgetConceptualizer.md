## Widget Conceptualizer

Widget conceptualizer is another way in which you can interact with the concept connection system. Widget conceptualizer is a visual way of creating applications which have data bound to it from the start. The way we are trying to move forward is that we allow creating universal widgets that can be used anywhere inside of any software but all of those data will be stored in our data fabric. 

Widget conceptualizer does not allow directly accessing functions but you need to allow it using tsccs library.
So if you want to call any function that has been called in this documentation then you must prefix it with "tsccs." .

Widget Conceptualizer is a more visual application so we have videos describing it.
